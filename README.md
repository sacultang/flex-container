# flex

```css
 .container {
   display:flex;
 }
```
수평 정렬 됨
<br>
- Flex Container
  - display:flex 값이 들어 있는것
<br>
- Flex items
  - container의 자식 요소들

### 속성
Flex Container | Flex Items
:--:| :--:
display | <b>order
flex-flow | flex
<b>flex-direction | <b>flex-grown
<b>flex-wrap | <b> flex-shirink
<b>justify-content | <b>flex-basis
algn-content | align-self
<b>algn-items |

flex-wrap | 속성
:--:| :--:
nowrap | 묶음(줄바꿈)없음
wrap | 여러 줄로 묶음
